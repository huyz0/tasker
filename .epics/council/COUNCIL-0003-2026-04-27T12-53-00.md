# Next Epic Recommendation

## What
**External Repository Integration (GitHub/Bitbucket)**
Implement management of external code repositories (GitHub, Bitbucket Cloud) with read-only access to PRs, set up authentication/linking, and manage project-to-repository relationships.

## Why
With Phase 1 (MVP) including Universal Search and core integrations complete, the council strongly recommends advancing to Phase 2 with External Repository Integration (Score: 4.78/5.00).

- **Roadmap Alignment**: Directly addresses the top item in Phase 2: Post-Launch.
- **User Value**: Allows developers and agents to see external PRs directly within the context of Tasker projects, unifying project management and code delivery.
- **Dependency Order**: Prerequisite for tracking builds and deployments on the repository level.
- **Technical Debt**: None related to this, but utilizing existing established ConnectRPC infrastructure will keep technical complexity low.
- **Implementation State**: Core entities (Projects, Tasks) are complete.
